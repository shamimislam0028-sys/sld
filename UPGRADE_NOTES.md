# UPGRADE — DOCX parser now works with ANY Word file (v201.1)

## যা বদলেছে

আগে DOCX ইম্পোর্ট শুধু একটা নির্দিষ্ট ফাইলের (`cos2A1.docx`) কনভেনশনেই কাজ করত।
পার্সার এখন **format-agnostic** — যেকোনো MCQ Word ফাইল ধরে।

দুটো ফাইল বদলানো হয়েছে (লজিক এক):
- `api/parse-docx.js`  ← Vercel সার্ভ করে এটাই (production path)
- `backend/server.js`  ← লোকাল ডেভ সার্ভার (LaTeX ম্যাথ সহ)

## এখন যেসব ভ্যারিয়েশন কাজ করে

| আগে (শুধু cos2A1) | এখন (যেকোনো ফাইল) |
|---|---|
| প্রশ্ন চিনত শুধু Word অটো-নাম্বার (`<w:numPr>`) | অটো-নাম্বার **অথবা** টাইপ-করা `১।` `1.` `1)` `(1)` `[1]` `Q1` `Question 1` |
| অপশন শুধু ল্যাটিন `A-D` | `A-D` + বাংলা `ক/খ/গ/ঘ` + বাংলা ডিজিট `১-৪` |
| প্রতি লাইনে একটা অপশন | এক লাইনে একাধিক অপশনও (`A. .. B. .. C. ..`) |
| উত্তর শুধু `উত্তর: B.` (কড়া) | `উত্তর` / `সঠিক উত্তর` / `Answer` / `Ans` / `Correct` + `B. ৩৪` টাইপ ট্রেলিং টেক্সটও |
| ম্যাথ প্রশ্ন/অপশন খালি হয়ে বাদ পড়ত | ম্যাথ টেক্সট ধরা পড়ে (ভগ্নাংশ `3/5`, রুট `√10`, পাওয়ার `p^2`) |
| উদ্দীপক/প্যাসেজ | আগের মতোই কাজ করে |

## টেস্ট রেজাল্ট
- `mcq_-_Copy.docx` (টাইপ-করা নাম্বার, w:br অপশন) → **১৫/১৫ MCQ**
- `cos2A1.docx` (অটো-নাম্বার, ম্যাথ) → **৩৫ MCQ**, সব ৪ অপশন + উত্তর ধরা পড়ে

## ডিপ্লয়

এই zip-এ `node_modules` নেই (Vercel নিজে ইনস্টল করে)।

**Vercel:** রিপো/ফোল্ডার push বা import করুন → auto-deploy। কিছু বদলাতে হবে না।

**লোকাল টেস্ট:**
```bash
npm install          # রুটে
npm run dev          # অথবা backend: node backend/server.js
```
