# UPGRADE — যেকোনো Word ফাইল → AI দিয়ে JSON MCQ (v201.2)

## যা হলো
DOCX ইম্পোর্ট এখন **Google Gemini (free)** দিয়ে যেকোনো ফরম্যাটের MCQ বের করে।
ফ্লো: `docx → টেক্সট বের করা → Gemini → পরিষ্কার JSON → pptx জেনারেটর`।

- **Primary:** প্রতিটা ফাইল Gemini দিয়ে পার্স হয় (সবচেয়ে নির্ভরযোগ্য)।
- **Safety-net:** key না থাকলে বা ফ্রি কোটা শেষ হলে অ্যাপ ভাঙে না — তখন বিল্ট-ইন
  রোবাস্ট রেগেক্স-পার্সারে নিজে থেকে ফিরে যায়।

বদলানো/নতুন ফাইল:
- `api/gemini-extract.js`  ← নতুন, শেয়ার্ড Gemini মডিউল
- `api/parse-docx.js`      ← Gemini primary + regex fallback (Vercel এটাই সার্ভ করে)
- `backend/server.js`      ← একই লজিক (লোকাল ডেভ)

কোনো নতুন npm ডিপেন্ডেন্সি লাগে না (Node-এর বিল্ট-ইন `fetch` ব্যবহার করে)।

## ★ যা করতেই হবে: Gemini API key বসানো

1. https://aistudio.google.com/apikey  → **Create API key** (ক্রেডিট কার্ড লাগে না)।
2. **Vercel-এ:** প্রজেক্ট → Settings → Environment Variables → যোগ করুন:
   - `GEMINI_API_KEY` = আপনার key
   - (ঐচ্ছিক) `GEMINI_MODEL` = `gemini-2.5-flash`  (Google মডেল রিনেম করলে এখানে বদলাবেন)
   → Save → **Redeploy**।
3. **লোকাল টেস্ট:** রুটে একটা `.env` ফাইলে `GEMINI_API_KEY=...` রাখুন, তারপর `npm run dev`।

key না বসালেও অ্যাপ চলবে — শুধু রেগেক্স-পার্সারে চলবে (আগের মতো)।

## ফ্রি টিয়ার (মাঝ-২০২৬)
- Gemini Flash ফ্রি টিয়ার: দিনে ~১,৫০০ রিকোয়েস্ট, কনটেক্সট ১০ লাখ টোকেন — একটা গোটা
  প্রশ্নব্যাংক এক কলেই যায়। মিনিটে ~১৫ রিকোয়েস্টের লিমিট আছে।
- খুব বড় ফাইলে (১০০+ প্রশ্ন) আউটপুট লিমিটে আটকালে ফাইলটা ভাগ করে দিন।
- ফ্রি টিয়ারে Google প্রম্পট ট্রেনিংয়ে ব্যবহার করতে পারে — MCQ পার্সিংয়ে সাধারণত সমস্যা নয়।

## রেসপন্স
`/api/parse-docx` আগের মতোই `{ mcqs: [...] }` দেয় (সাথে debug-এর জন্য `source` ফিল্ড:
`gemini` / `regex-fallback` / `regex-no-key`)। ফ্রন্টএন্ডে কিছু বদলাতে হয়নি।

## ডিপ্লয়
zip-এ `node_modules` নেই — Vercel নিজে ইনস্টল করে। ফোল্ডার push/import → env var বসান → deploy।
